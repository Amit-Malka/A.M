# Amit Malka - AI Developer Portfolio

Professional portfolio showcasing AI development expertise, projects, and experience.

## Features

- **Responsive Design** - Works on all devices
- **Modern Tech Stack** - React 19 + TypeScript + Tailwind CSS
- **Enhanced Projects** - Detailed case studies with interactive data
- **Sophisticated Animations** - Smooth interactions via Framer Motion
- **SEO Optimized** - Static JSON-LD schema markup in HTML head

## Tech Stack

- **Frontend**: React 19, TypeScript, Tailwind CSS
- **Animations**: Framer Motion
- **Build Tool**: Create React App
- **Deployment**: Vercel

## Project Structure

```
src/
├── components/     # React components
├── utils/          # Utilities (projectData, downloadCV)
├── App.tsx         # Main app
└── index.tsx       # Entry point

public/             # Static assets (favicon, images, CV)
```
